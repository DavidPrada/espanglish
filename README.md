#Espanglish keyboard layout
A modified English(US) keyboard layout compatible with Spanish(ES) symbols

##Layout
![alt tag](https://raw.github.com/DavidPrada/espanglish/master/espanglish.png)

##How to type the spanish symbols
```
á = ´ then a
é = ´ then e
í = ´ then i
ó = ´ then o
ú = ´ then u
ñ = Shift + ´ then n
ü = AltGr + : then u
€ = AltGr + e
® = AltGr + r
™ = AltGr + t
º = AltGr + o
ª = AltGr + a
` = AltGr + '
¿ = AltGr + /
```
##Usage
`sudo bash espanglish.bash`

`Menu > Preferences > Keyboard > Layouts > [+] > search for "espanglish" > Add`
